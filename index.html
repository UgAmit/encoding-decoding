Encoding-Decoding Repository

A collection of encoding and decoding algorithms and implementations.

Introduction

This repository provides implementations of various encoding and decoding algorithms. These algorithms are essential for data representation, compression, and security.

Base64 Encoding/Decoding

Base64 is a widely used encoding scheme that converts binary data into an ASCII string format.

Example (JavaScript)


const base64Encode = (str) => btoa(str);
const base64Decode = (str) => atob(str);

const originalString = "Hello, World!";
const encodedString = base64Encode(originalString);
const decodedString = base64Decode(encodedString);

console.log("Original:", originalString);
console.log("Encoded:", encodedString);
console.log("Decoded:", decodedString);

URL Encoding/Decoding

URL encoding, also known as percent-encoding, is used to encode characters in a URL that have special meaning.

Example (JavaScript)


const urlEncode = (str) => encodeURIComponent(str);
const urlDecode = (str) => decodeURIComponent(str);

const originalURL = "https://example.com/?q=search term&space=  ";
const encodedURL = urlEncode(originalURL);
const decodedURL = urlDecode(encodedURL);

console.log("Original:", originalURL);
console.log("Encoded:", encodedURL);
console.log("Decoded:", decodedURL);
